env PYTHON_CONFIGURE_OPTS="--enable-shared" LD_LIBRARY_PATH=~/.pyenv/versions/3.6.5/lib/ pyenv install -k 3.6.5

2.6 补充

上面这些只是网络上找到的方法，官方文档中也给出了一些的替换方法，如PyImport_Import可以改为PyImport_ImportModule，参数改为字符串即可，pArgs也可以通过Py_BuildValue来设置
