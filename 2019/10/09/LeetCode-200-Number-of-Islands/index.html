小岛的数量

题目描述：输入一个为二维的格子，格子中'1'代表岛，'0'代表水，计算格子中小岛的数量。被水包围的才算岛，假定格子的四边都是水。

深度优先搜索

这题本质上是求矩阵中连续区域的个数，只需要从'1'开始找到所有的连通区域即可。时间复杂度O(MN)，空间复杂度O(1)

这题还可以用一个visited数组来代替grid[i][j] = '0'作为该位置的数据已经访问过的标志，其他的思路一致。时间复杂度仍然是O(MN)，空间复杂度O(MN)

广度优先搜索

借助队列实现，如果遇到一个'1'就搜索周围的区域。时间复杂度O(MN)，空间复杂度O(MN)

这里还可以做一个优化，visited变量可以从二维降成一维，每个坐标的表示可以写为pos = x * n + y，翻过来可以通过x = pos / n，y = pos % n获取原始坐标
