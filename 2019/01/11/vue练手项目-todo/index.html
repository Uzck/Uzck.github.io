Vue练手项目 Todo App

发表于 2019-01-11 | 分类于 前端

过了一遍Vue的教程(没有js基础，纯瞎看)写了个没有css丑到爆的todo(还只有添加没删除的屑)

项目用vue-cli生成，使用到的自定义组件有三个

AddButton.vue

用于输入新的todo

input绑定newTodo变量，button绑定_click方法，在_click的实现里可以看到用的this.$emit方法把输入的值传到父组件Todo.vue中，并且父组件监听add-todo属性，把数据添加到todo-list中

TodoList.vue

TodoList导入子组件TodoItem，同时TodoList作为Todo的一个子组件，接受Todo传入的sources，并把每个source传入TodoItem

TodoItem.vue

todo-item很简单，只是显示字符串

Todo.vue

Todo.vue导入AddButton和TodoList两个组件，@addTodo=getDataFromInput是上文提到了的用来监听AddButton子组件传出来的值，具体的看getDataFromInput实现，`this.source获取原来的数据列表，$event``是AddButton里传出来的值，这里只是简单的添加。

router.js

这块没了解过，照猫画虎

这样在程序跑起来之后在浏览器输入http://localhost:8000/todo就可以访问Todo.vue页面了

总结

这里面涉及的知识点:

  1. v-bind:value以及:value
  2. v-for的使用
  3. v-on:method以及@method
  4. 父组件与子组件之间传值：子组件获取父组件的数据需要设置props属性，父组件从子组件获取数据需要子组件调用$emit方法
