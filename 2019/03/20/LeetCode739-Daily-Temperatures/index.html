LeetCode739 Daily Temperatures

发表于 2019-03-20 | 分类于 数据结构与算法

题目描述

给出连续若干天的气温数据如[73, 74, 75, 71, 69, 72, 76, 73]，输出多少天后气温会高于当天，上面例子的输出结果为[1, 1, 4, 2, 1, 1, 0, 0]。温度的范围从30到100，数据长度1到30000。

每次看到别人的解法就觉得自己用那么长时间写了那么一大堆像个智障。

按样例输入[73, 74, 75, 71, 69, 72, 76, 73]

初始时：stack=[]

i = 0：stack为空，stack插入0，此时stack=[0]

i = 1：temperatures[1] > temperatures[0]，ret[0] = 1 - 0，stack=[1]

i = 2：temperatures[2] > temperatures[1]，ret[1] = 2 - 1，stack=[2]

i = 3：temperatures[3] < temperatures[2]，stack=[2,3]

i = 4：temperatures[4] < temperatures[3]，stack=[2,3,4]

i = 5：temperatures[5] > temperatures[4]，ret[4] = 1, stack=[2, 3]，temperatures[5] > temperatures[3]，ret[3] = 2, stack=[2]， stack.push(5)，stack=[2,5]

i = 6：temperatures[6] > temperatures[5]，ret[5] = 1，stack=[2]，temperatures[6] > temperatures[2]，ret[2] = 4，stack=[]，stack.push(6)，stack=[6]

i = 7：temperatures[7] < temperatures[6]

循环结束：这里用到了java数组的一个特性，创建数组没有手动复制的话默认值为0
