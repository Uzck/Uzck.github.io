LeetCode841 Keys and Rooms

发表于 2019-03-27 | 分类于 数据结构与算法

题目描述

每个房间里有若干钥匙，钥匙能开对应房间的门，在0号房间不需要钥匙的情况下判断能否打开所有房间。

输入[[1],[2],[3],[]]，表示0号房间有一号房间的钥匙，1号房间有2号房间的钥匙，2号房间有3号房间的钥匙，3号房间没有钥匙。

广度优先遍历解法

思路是广度优先遍历外加额外的数组记录房间是否已经打开过，也可以用递归方法来写

用字典实现

用dict或Set数据结构可以减少同一个房间的多个钥匙的遍历次数
