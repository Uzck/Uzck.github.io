LeetCode654 Maximum Binary Tree

发表于 2019-03-10 | 分类于 数据结构与算法

题目描述

输入是一个无重复数据的一维数组:[3, 2, 1, 6, 0, 5]， 构建一个二叉树，根节点保存数组里最大的值，最大值左边的部分用来构建左子树，右边的部分构建右子树，也就是说[3,2,1]构建左子树，[0,5]构建右子树。

递归

递归也是很正常的思路

很简单，每次找出该区间的最大值，再分别找出两边区间的最大值。

Stack解法

Discussion里面提到的解法，贴下python版本

算法时间复杂度O(n)，比递归的$O(n^2)$快很多。
